# 🔢 Vishingh Calculator - All-in-One Utility Tool

**Vishingh Calculator** is a premium, modern, and high-performance utility application developed by **VishinghSoft**. It goes beyond basic arithmetic, offering specialized tools for daily life calculations.

---

## 🚀 Key Features

- **Standard Calculator**: Sleek Neumorphic design with full mathematical operator support.
- **BMI Calculator**: Quickly calculate Body Mass Index for health monitoring.
- **EMI Calculator**: Plan your loans by calculating Monthly Installments easily.
- **GST Calculator**: Calculate total amount with GST in one tap.
- **Unit Converter**: Seamlessly convert distances (KM to Meters).
- **Dark & Light Mode**: Persistent theme support that saves your preference.
- **Fully Responsive**: Optimized for both mobile and desktop screens.
- **Keyboard Support**: Full integration for desktop users.

---

## 🛠️ Technical Stack

- **Frontend**: HTML5, CSS3 (Custom Styles)
- **Design System**: Neumorphic UI / Material Poppins Font
- **Logic**: Vanilla JavaScript (ES6)
- **Deployment Ready**: Lightweight and fast loading.
