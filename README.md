# 🔢 Vishingh Calculator - All-in-One Utility Tool

![VishinghSoft Logo](vs.png)

**Vishingh Calculator** is a premium, modern, and high-performance utility application developed by **VishinghSoft**. It goes beyond basic arithmetic, offering specialized tools for daily life calculations with a sleek Neumorphic UI.

---

## 🚀 Key Features

* **Standard Calculator:** Sleek design with full mathematical operator support.
* **BMI Calculator:** Quickly calculate Body Mass Index for health monitoring.
* **EMI Calculator:** Plan your loans by calculating Monthly Installments easily.
* **GST Calculator:** Calculate total amount with GST in one tap.
* **Global Currency Converter:** Live exchange rates for 160+ countries with flags.
* **Unit Converter:** Seamlessly convert distances (KM to Meters, etc.).
* **Dark & Light Mode:** Persistent theme support that saves your preference.
* **Fully Responsive:** Optimized for mobile, tablets, and desktop screens.

---

## 🛠 Technical Stack

* **Frontend:** HTML5, CSS3 (Custom Properties for Themes)
* **Design System:** Neumorphic UI / Material Design Hybrid
* **Typography:** Poppins (Google Fonts)
* **Logic:** Vanilla JavaScript (ES6+)
* **API Integration:** ExchangeRate-API for real-time currency data.
